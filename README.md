# DelayedJobScheduler

DelayedJobScheduler provides a service where people schedule delayed job with prize and winning bidder can get reward on execution.

## How To

Compile: yarn compile

Testing: yarn test

Code Coverage: yarn coverage

Deploy: yarn deploy
